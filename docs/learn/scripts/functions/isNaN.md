# Function: isNaN

Returns True / False if the displayed value is integer.

Overloadings:
+ ``isNaN(number)``

Example use:
```
var input = input("Type your anything: ");
if isNaN(input) ->> {
    print("Is number!")
};

#>
OUTPUT

Type your name: <input> # 123
Is number!
-----
Type your name: <input> # abc
<#
```
