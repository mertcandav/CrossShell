# netinfo

This module is a module that allows you to get information about the network.

### Parameters

+ ``-help``<br>
    Show help of module. This parameter can only be used alone.

+ ``-all``<br>
    Show all informations. This parameter can only be used alone.

+ ``-hostname``<br>
    Show hostname.

+ ``-ip``<br>
    Show ip address.
